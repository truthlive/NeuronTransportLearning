1. meshgeneration: matlab code to generate hexahedral mesh given input neuron skeleton.

2. preprocess: code used to create batch jobs to run IGA simulation on super computer. Once the input mesh is ready, run IGA solver in https://github.com/CMU-CBML/NeuronTransportIGA to get simulation data.

3. postprocess: code used to postprocess IGA simulation data and generate the input files for the GNN model.
